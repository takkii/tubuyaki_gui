# Tubuyaki_GUI

RubyGemsにあるtubuyakiとGithubで管理しているtubuyakiをもう少し改良したtubuyaki_guiです。

変更点

>log/backup.txtにつぶやきの履歴を保存するようにしたところです

>logフォルダとbackup.txtは自動で作成されます

かなり使いやすくなりました。

***日本語で25文字〜35文字位が目安です***

ToDo管理にお使いくださいませ。

詳細

backup.txtのファイル内エンコーディングは、

>SJIS 指定です。

なぜなら、zinbei,zinbeijettのためです。
